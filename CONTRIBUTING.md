# Contributing to DeepVision-Tracking

We welcome contributions to improve the AI Surveillance Dashboard!

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally.
3. **Create a new branch** for your feature or bugfix.
4. **Commit your changes** with clear, descriptive messages.
5. **Push to your branch** and **create a Pull Request**.

## Technical Standards

- Use **Type Hints** for all new functions.
- Follow **PEP 8** coding standards.
- Update documentation if you add new features.
- Ensure all tests pass before submitting.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the GitHub repository.

Thank you for helping improve AI-based surveillance technology!
